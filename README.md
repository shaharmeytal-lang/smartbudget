# 💰 SmartBudget

SmartBudget היא אפליקציית ניהול תקציב אישית שמאפשרת למשתמשים לעקוב אחרי הכנסות והוצאות בזמן אמת, לנתח את ההתנהלות הפיננסית שלהם ולקבל תמונה ברורה של מצב החשבון שלהם.

---

## 🚀 סקירה כללית

האפליקציה מאפשרת:
- הוספת הכנסות והוצאות
- עריכה ומחיקה של עסקאות
- סינון לפי חודש
- חיפוש עסקאות
- חישוב אוטומטי של יתרה

המערכת בנויה כ־Full Stack עם Frontend מודרני ו־Backend בענן.

---

## ❗ איזו בעיה הפרויקט פותר

רוב האנשים:
- לא יודעים לאן הכסף שלהם הולך
- מנהלים הוצאות בצורה לא מסודרת (אקסל / זיכרון / פתקים)
- לא מקבלים תמונה ברורה של התקציב החודשי

SmartBudget פותרת את זה על ידי ריכוז כל הנתונים במקום אחד עם חישוב אוטומטי של מצב פיננסי.

---

## 🎯 קהל יעד

האפליקציה מיועדת ל:
- סטודנטים
- עובדים שכירים
- אנשים שרוצים שליטה בהוצאות שלהם
- משתמשים שמחפשים פתרון פשוט במקום אקסל

---

## 🆚 מתחרים ובידול

### מתחרים:
- Excel / Google Sheets
- מעקב ידני (פתקים / מחברת)
- אפליקציות בנקאיות בסיסיות
- אפליקציות תקציב מסובכות מדי

### הבידול של SmartBudget:
- ממשק פשוט ומהיר
- הזנת נתונים בזמן אמת
- סינון לפי חודש בלחיצה אחת
- חיפוש מהיר
- חיבור לענן באמצעות Supabase
- ממשק נקי ללא עומס מיותר

---

## 🧠 טכנולוגיות

- Next.js (React)
- TypeScript
- Tailwind CSS
- Supabase (Database & Backend)
- Vercel (Deployment)
